Diagrama Bow Tie

Módulo de análise de riscos — baseado na ABNT NBR ISO/IEC 31010:2012, Anexo B.21

Arraste um elemento da barra lateral para o canvas para criá-lo. Duplo clique em qualquer bloco para editar seus dados. Clique no "×" de um bloco para removê-lo instantaneamente. Arraste um bloco para movê-lo livremente. Segure Espaço + clique (ou o botão do meio do mouse, ou uma área vazia) para navegar pela tela. Role o mouse para percorrer o canvas; Ctrl + roda do mouse (ou pinça no trackpad) para zoom no ponto do cursor.
